Trabajo final de la 1ª Evaluación

HTML+CSS

Profesor: Angel Martínez Cañadas

Proyecto: La web de mi empresa ó...

El objetivo del proyecto es elaborar un sitio web empresarial o personal, siguiendo una metodología basada en estándares.

Requisitos

  • Se escribirá en HTML5 y CSS. Está prohibido utilizar etiquetas discontinuadas como <font> o <frame> y aplicar todos los estilos mediante CSS.
  • Las páginas deberán ser válidas. Se asegurará que http://validator.w3.org no nos devuelve ningún error.
  • La estructura del sitio será homogénea

Requisitos II

  • Al menos 4 páginas.
  • Navegación entre las páginas del sitio.
  • Tendrán una hoja de estilos css, aunque sea sencilla.
  • Una de las páginas tendrá un formulario (p.e. formulario de contacto).
  • Se incluirán imágenes.
  • Tendrá al menos una tabla de datos.

Se valorará

  • Cumple con los requisitos mínimos.
  • Logo.
  • Navegación.
  • Contenido relevante.
  • Encabezados, pies.
  • Usabilidad, accesibilidad.
  • Responsive.

Ejemplos

Algunos ejemplos de páginas sencillas, similares a lo que se pide

  • Ars Docendi
  • GWT (con un menú más sencillo)
  • GSWD
  • Con un poco más de maquetación: pelao.org
